## 🚀 Instalação Rápida

⚡ **Em menos de 1 minuto você já estará rodando localmente.**

1. Baixe o executável mais recente na [página de releases](https://github.com/forgesystems-sete/DistroHub.Community/releases).
2. Execute \`DistroHub.Community.exe\` (Windows) ou \`./DistroHub.Community\` (Linux/macOS).
3. Acesse \`http://localhost:5000\` no seu navegador.

> 💡 Alternativa: execute via CLI com \`dotnet run\` após clonar o repositório.

**Pré-requisitos:** Nenhum. O executável é auto-contido (inclui o runtime .NET 8).
